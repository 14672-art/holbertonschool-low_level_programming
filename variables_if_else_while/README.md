# Variables, if, else, while

Ce projet contient des scripts et programmes C permettant de comprendre l'utilisation des variables, des structures conditionnelles (if, else) et des boucles (while) en C89.

## Description des fichiers
* **0-positive_or_negative.c** : Programme qui attribue un nombre aléatoire à une variable et affiche si ce nombre est positif, négatif ou nul.
* **1-last_digit.c** : Programme qui récupère le dernier chiffre d'un nombre aléatoire et indique s'il est supérieur à 5, égal à 0, ou inférieur à 6 et non nul.
* **2-print_alphabet.c** : Programme qui affiche l'alphabet complet en minuscules suivi d'un saut de ligne, en utilisant uniquement deux fonctions putchar.
